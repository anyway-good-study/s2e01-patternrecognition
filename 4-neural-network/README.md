- 설명

위 출력결과에서 [[ bias, w0, w1 ]] 이 학습되는 과정을 볼 수 있다.

input/linear-reg-train.txt 학습 데이터를 참고하여 b(bias 는 1), x0, x1 값을 대입하여 확인할 수도 있다.

y = w0*x0 + w1*x1 + b (단, w0=1.0, w1=1.0, b=1.0)

위 함수를 만족하는 W를 구한 것을 알 수 있다.
